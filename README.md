## Install
<ul>
<li>git clone https://github.com/NikitaPoplavskiy/JsonApi.git</li>
<li>cd JsonApi</li>
<li>composer update</li>
<li>cp .env.example .env</li>
<li>php artisan key:generate</li>
<li>php artisan migrate</li>
<li>php artisan serve</li>
</ul>
