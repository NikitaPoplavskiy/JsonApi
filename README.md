## Install
<li>
<ul>git clone https://github.com/NikitaPoplavskiy/JsonApi.git</ul>
<ul>cd JsonApi</ul>
<ul>composer update</ul>
<ul>cp .env.example .env</ul>
<ul>php artisan key:generate</ul>
<ul>php artisan migrate</ul>
<ul>php artisan serve</ul>
</li>
